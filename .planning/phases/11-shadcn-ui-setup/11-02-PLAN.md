<task type="auto">
  <name>Task 2: Configure Poppins + Inter fonts</name>
  <files>src/app/layout.tsx, src/app/globals.css</files>
  <action>
1. Update src/app/layout.tsx to load fonts via next/font/google:

```tsx
import { Poppins, Inter } from "next/font/google"

const poppins = Poppins({
  subsets: ["latin"],
  weight: ["400", "500", "600", "700"],
  variable: "--font-poppins",
})

const inter = Inter({
  subsets: ["latin"],
  variable: "--font-inter",
})
```

2. Apply font variables to the html/body element:

```tsx
<html lang="en" className={`${poppins.variable} ${inter.variable}`} suppressHydrationWarning>
```

3. Add font-family definitions to globals.css @theme inline block:

```css
@theme inline {
  /* Existing color mappings... */

  --font-sans: var(--font-inter), ui-sans-serif, system-ui, sans-serif;
  --font-heading: var(--font-poppins), ui-sans-serif, system-ui, sans-serif;
}
```

4. Add base typography styles in @layer base:

```css
@layer base {
  * {
    @apply border-border outline-ring/50;
  }
  body {
    @apply bg-background text-foreground font-sans;
  }
  h1, h2, h3, h4, h5, h6 {
    @apply font-heading;
  }
}
```

**What to avoid:**
- Do NOT use @import for Google Fonts (next/font is more performant)
- Do NOT add font weights we don't need (bloats bundle)
  </action>
