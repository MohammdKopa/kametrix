**Key constraints from research:**
- Use `motion` package (not `framer-motion`) - rebranded in 2024
- Import from `"motion/react"` (not old path)
- Use `useScroll` + `useTransform` for scroll-linked effects
- Keep durations 0.4-0.7s max
- Use small Y offsets (20-40px) to avoid layout shift

**User vision:**

- Glow should create depth and draw the eye

  <name>Task 2: Enhance Hero with scroll-linked glow animation</name>
  <files>src/components/marketing/Hero.tsx</files>
  <action>
    Convert Hero.tsx to use Motion for scroll-linked glow effect:

    1. Add `"use client"` directive (required for hooks)
    2. Import from `motion/react`: `motion, useScroll, useTransform`
    3. Add `useScroll()` hook to get `scrollYProgress`
    4. Create transforms for the radial glow:
       - `glowOpacity`: Map scrollYProgress [0, 0.3] to opacity [0.2, 0]
       - `glowScale`: Map scrollYProgress [0, 0.3] to scale [1, 0.7]
    5. Convert the main radial glow div to `motion.div` with:
       - `style={{ opacity: glowOpacity, scale: glowScale }}`
    6. Keep the secondary glow static (don't over-animate)
    7. Add subtle entrance animation to hero content:
       - `initial={{ opacity: 0, y: 20 }}`
       - `animate={{ opacity: 1, y: 0 }}`
       - `transition={{ duration: 0.6, ease: "easeOut" }}`

    **Pattern from RESEARCH.md:**
    ```tsx
    const { scrollYProgress } = useScroll();
    const glowOpacity = useTransform(scrollYProgress, [0, 0.3], [0.2, 0]);
    const glowScale = useTransform(scrollYProgress, [0, 0.3], [1, 0.7]);
    ```

    **What to avoid:**
    - Don't animate EVERY element - just the glow and main content container
    - Don't use large Y offsets (keep to 20px)
    - Don't slow down durations past 0.7s
  </action>
